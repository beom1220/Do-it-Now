# [1676] 팩토리얼 0의 개수

---

## 📌 **Algorithm**

---

수학

## 📍 **Logic**

---

```
for (int i = 1; i <= n; i++) {
		while (true) {
        if (i % 2 == 0) {
            i /= 2;
            two++;
        } else {
            break;
        }
    }

		while (true) {
        if (i % 5 == 0) {
            i /= 5;
            five++;
        } else {
            break;
        }
    }
}
```

- 팩토리얼을 직접 계산하지 않고, 인수분해했을 때의 2와 5의 개수만 따로 계산하였음.
- 2와 5가 각각 1개 이상이라면, 둘 중 적게 나온 숫자의 개수를 출력.
- 둘 중 하나라도 0개라면 0을 출력하게끔 하였음.
